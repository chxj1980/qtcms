包含文件

#include <IAudioPlayer.h>

 

函数定义

virtual int SetMainWnd(int nWnd) = 0;
设置播放窗口。

virtual int SetAudioParam(int nChannel,int nSampleRate,int nSampleWidth) = 0;
设置声道(nChannel)， 采样率(nSampleRate)，采样位宽(nSampleWidth)。

virtual int SetVolume(unsigned int uiPersent) = 0;
设置播放的音量。

virtual int Play(char *pBuffer,int nSize) = 0;
从缓冲区读取帧解码并播放。

virtual int EnablePlay(bool bEnabled) = 0;
声音播放开关, bEnabled为true,声音打开,反之关闭。

virtual int GetPlayStatus() = 0;
获取当前声音开头状态。

virtual int Stop() = 0;
关闭声音播放,并释放设备资源。
