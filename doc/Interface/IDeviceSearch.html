包含文件
#include<IDeviceSearch.h>

函数定义
virtual int Start() = 0;
开启搜索设备，搜索结构以事件的形式抛出。
返回值_enErrorCode 代码
OK：进行检索。E_SYSTEM_FAILED：检索失败

virtual int Stop() = 0;
停止搜索设备。
返回值_enErrorCode 代码
OK：停止检索。E_SYSTEM_FAILED：停止失败

virtual int Flush() = 0;
重新搜索设备。
返回值_enErrorCode 代码
OK：调用成功。E_SYSTEM_FAILED：调用失败。

virtual int setInterval(int nInterval) = 0;
设置检索每次发送检索请求的间隔
返回值_enErrorCode 代码
OK：设置成功。E_INVALID_PARAM：参数错误。E_SYSTEM_FAILED：设置失败

virtual IEventRegister * QueryEventRegister() = 0;

enum _enErrorCode{
OK,
E_INVALID_PARAM,
E_SYSTEM_FAILED,
};

备注：以上的Start（）和Stop两个接口为非阻塞实现，获取搜索的结构以注册事件来得到
