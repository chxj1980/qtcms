接口简介：
名称：local player组件同步回放接口文档
功能：提供本地录像文件的同步回放接口
包含头文件：ILocalPlayer.h
接口定义：
@1
简介：搜索在限定时间范围内的本地录像文件
输入参数：
start：开始时间（时间格式："yyyy-MM-dd hh:mm:ss"）
end：结束时间（时间格式："yyyy-MM-dd hh:mm:ss"）
filelist：返回搜索到的文件list（完整的文件路径）
返回值：
0：成功
1：失败
virtual int SearchLocalPlayFileList(const QDateTime &start,const QDateTime &end,QStringList &filelist)=0;

@2
简介：添加播放文件到同步组中，并指定文件播放的窗口ID
输入参数：
filelist:文件名列表，列表中的文件必须是同一个通道号下的文件，包括完整的文章路径，如：F:\project\date1\devname

\chl2\filename.avi，F:\project\date2\devname\chl2\filename.avi
wnd:播放窗口的id
start:开始播放时间(时间格式："yyyy-MM-dd hh:mm:ss")
end：结束播放时间(时间格式："yyyy-MM-dd hh:mm:ss")
返回值：
0：添加成功
1：添加失败，通道组已经满
2：添加失败，窗口已经被占用
3：添加失败，（失败原因未定义）
virtual int AddFileIntoPlayGroup(QStringList const filelist,QWidget *wnd,const QDateTime &start,const QDateTime &end)=0;

@3
简介：设置同步组数量
输入参数：
num:设置同步组的数量值
返回值：
0：设置成功
1：设置失败
virtual int SetSynGroupNum(int num)=0;

@4
简介：组播放
返回值：
0：调用成功
1：调用失败
virtual int GroupPlay()=0;

@5
简介：暂停播放
返回值：
0：调用成功
1：调用失败
virtual int GroupPause()=0;

@6
简介：继续播放
返回值：
0：调用成功
调用失败
virtual int GroupContinue()=0;

@7
简介：停止播放
返回值：
0：调用成功
1：调用失败
virtual int GroupStop()=0;

@8
简介：快进
输入参数：
speed：正常播放速度的倍数（可选值：2，4，8）
返回值：
0：调用成功
1：调用失败
virtual int GroupSpeedFast(int speed)=0;

@9
简介：慢放
输入参数：
speed：正常播放速度的1/N倍（可选值：2，4，8）
返回值：
0：调用成功
1：调用失败
virtual int GroupSpeedSlow(int speed)=0;

@10
简介：恢复正常的播放速度
返回值：
0：调用成功
1：调用失败
virtual int GroupSpeedNormal() = 0;
