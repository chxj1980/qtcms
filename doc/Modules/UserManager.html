用户信息存储位置：
system.db

用户表
user_infomation

id integer primary key autoincrement
username text
password text
level integer
mask1 integer
mask2 integer

password存储的是经过MD5编码的文本
level是用户权限等级，取值范围为0~3，分别为超级管理员、管理员、用户和游客四个等级。系统中仅存在一个超级管理员，即admin用户
mask1,maks2为用户详细权限，现阶段该值保留，统一区0xFFFFFFFF

表创建代码：

create table user_infomation(
id integer primary key autoincrement,
username text,
password text,
level integer,
mask1 integer,
mask2 integer
);
