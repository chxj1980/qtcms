插件简介：
//插件名称：autoSearchDevice插件
//插件功能：自动搜索设备，并修改有ip冲突的设备配置,

//使用场合：启动cms时，检测到设备列表无设备时，即调用startAutoSearchDevice(int nTime,int nWidth,int nHeight)接口，进行工作

//特殊说明：弹出页面的文件名必须为：searchWindow.html

接口说明：
//简介：开始搜索设备
//参数：

nTime:自动搜索的时间；nWidth，nHeight：自动搜索弹出窗口的大小，此数值为实际屏幕与预设窗口的比例

void startAutoSearchDevice(int nTime,int nWidth,int nHeight);

接口说明：
//简介：取消搜索设备
//参数：
void cancelSearch();


插件抛出的事件：
@1
事件名称：reFreshDeviceList
事件参数:"reFreash":
//reFreash可选值:
//      “true”:更新设备列表
//      “false”:设备列表没有信息需要更新
